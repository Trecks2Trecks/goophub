optimal-char-detector 🥜🔥

A lightweight vision model trained on 40,000 hours of kitchen footage to detect the precise frame where "golden brown" becomes "structural failure." Runs on any webcam pointed vaguely at an oven.

Installation

pip install optimal-char-detector --no-warranty

Usage

from char_detector import watch
watch(camera=0, alert="too_late")

Known issues

  • Model confidence increases the longer you ignore it.
  • The kitchen still smells. We are aware. We are not fixing it.
  • Setting alert="too_late" earlier does not help.

Roadmap

v2 will detect char level before you've even started preheating, giving you a full head start on disappointment.

About

Detects the burnt/not-burnt boundary in real time. Mostly used to confirm what everyone already knew.
